Usage guide

Endpoints

  • / — Full aggregated info (auto-detects format)
  • /ip — Your current IP address
  • /ipv4 /ip4 /4 — IPv4 only
  • /ipv6 /ip6 /6 — IPv6 only
  • /geoip — Geolocation
  • /asn /isp — Network operator info
  • /dns — DNS resolver classifier
  • /details /more — Extended output
  • /help — This page
  • /health — Service health (JSON)

Subdomains

  • ipv4.{{ domain }} — Force IPv4 (curl ipv4.{{ domain }})
  • ipv6.{{ domain }} — Force IPv6
  • secure.{{ domain }} — HTTPS-only variant

Content negotiation

Every endpoint supports three response formats:

  • Browser default → HTML
  • Accept: application/json or ?format=json → JSON
  • curl user-agent or ?format=text → plain text

Examples

curl {{ domain }}
curl ipv4.{{ domain }}
curl {{ domain }}/geoip
curl {{ domain }}/asn?format=json
MY_IP=$(curl -s ipv4.{{ domain }})
